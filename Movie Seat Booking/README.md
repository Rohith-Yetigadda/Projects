# Movie Seat Booking System

## Description
A fully functional movie seat booking web application built using **HTML, CSS, and JavaScript**.  
The app dynamically generates seats, handles user interaction, and maintains booking state on the client side without using any frameworks.

## Features
- Dynamic seat generation (Rows A–P, 20 seats per row)
- Realistic theatre layout with aisles (5–10–5 seat blocks)
- Tier-based pricing:
  - Silver – ₹180
  - Gold / Prime – ₹250
  - Platinum / Recliner – ₹330
- Interactive seat selection and booking
- Real-time price calculation
- Persistent booked seats using LocalStorage
- Reset booking functionality

## DOM Concepts Used
- Dynamic DOM creation and updates
- Event handling and delegation
- State-based UI rendering
- LocalStorage for persistence
